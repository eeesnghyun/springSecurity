# Spring Boot + Security 예제
⚙️️ **프로젝트 구조**
- - -
```
.
└── src/
    └── main/
        └── java/
            ├── springSecurity/
            │   ├── component            : Security 관련 인터페이스 구현체 관리
            │   ├── config               : Config 클래스 관리
            │   ├── user                 : 회원 관리
            │   ├── HomeController.java  : 화면 컨트롤러
            │   └── SpringSecurityApplication.java
            └── resources/
                ├── static               : 정적 자원 파일 관리
                ├── templates            : 뷰 템플릿 파일 관리(Thymeleaf)
                └── application.properties 
```
